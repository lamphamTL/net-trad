Netflix Custom Subtitles — v0.1

Tiny Chrome/Edge extension that loads your own .srt / .vtt file and renders it on top of Netflix while hiding the built-in subtitles.

Files in this folder

  • manifest.json — extension config
  • content.js — injects overlay into Netflix player
  • overlay.css — hides native subs + styles ours
  • popup.html / popup.js — toolbar UI to load file + set offset

Install in Firefox (temporary, recommended for dev)

  1. Open about:debugging#/runtime/this-firefox
  2. Click Load Temporary Add-on…
  3. Pick any file inside this folder (e.g. manifest.json).
  4. Extension appears in toolbar. Pin it.
Temporary add-ons get removed when Firefox restarts. Re-load after each browser restart. To make it persist permanently, the add-on must be signed by Mozilla (AMO) — overkill for personal use.

Alternative: persistent install (Firefox Developer Edition / Nightly only)

  1. Open about:config
  2. Set xpinstall.signatures.required to false
  3. Zip the folder contents (not the folder itself) → rename to .xpi
  4. Drag the .xpi onto Firefox window
Icon missing? Add any 128x128 PNG named icon.png in this folder, or delete the icons block in manifest.json.

Per-title scoping

Subtitles are stored per Netflix title ID (the number in netflix.com/watch/81234567). Each episode has its own ID. Uploading only affects the currently-playing title. Switching to another title shows its own subs (or nothing).

The popup shows Current title: 81234567 at top. Make sure a Netflix /watch/ tab is open before uploading.

Workflow: rip → translate externally → upload

  1. Go to netflix.com, start playing the episode.
  2. In Netflix subtitle menu, turn on subtitles in the language you want to translate from (e.g. English).
  3. Wait a few seconds for the player to fetch the subtitle file. Skip back a couple seconds if nothing captures.
  4. Open extension popup → Download captured subs (.srt) button enables → click it. SRT downloads.
  5. Translate the SRT externally:
    • DeepL (best quality) — paste in deepl.com/translator or use API
    • ChatGPT / Claude — paste contents, ask "translate to Vietnamese, keep timestamps untouched"
    • Google Translate — translate.google.com → Documents tab → upload .srt
    • subtitletranslator.com — drag SRT, pick language, download
  6. Save the translated .srt to disk.
  7. Back in extension popup → Choose file → pick translated .srt.
  8. Turn off / hide Netflix's own subs (CSS hides them anyway).
  9. Adjust Sync offset if needed (negative = subs earlier).
Translation tip: For best timing fidelity, translate the SRT line by line with timestamps preserved. ChatGPT/Claude do this well if you say "keep timestamps and blank lines exactly as-is, translate only the text".

Troubleshooting

  • Nothing shows up — Open DevTools console on the Netflix tab (F12), look for [CustomSubs] log lines. Confirm cue count > 0. Console scope must be set to the page, not the extension.
  • Firefox: extension gone after restart — Temporary add-ons are wiped. Re-load from about:debugging.
  • Firefox: errors about chrome.* — Firefox provides a compat shim, should just work. If not, replace chrome. with browser. in content.js and popup.js.
  • Native subs still visible — Netflix sometimes uses a different CSS class. Inspect the subtitle DOM and add its selector to overlay.css.
  • Overlay disappears after switching episodes — Should self-reattach via MutationObserver. If not, refresh page.
  • Fullscreen breaks — Overlay attaches to .watch-video which is the fullscreen container, should be fine. If not, file an issue with the current player markup.
